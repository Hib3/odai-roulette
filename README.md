# Blog Odai Roulette

GitHub Pages でそのまま公開できる、音あり・アニメーションありのブログのお題ルーレットです。

## Files

- `index.html`: アプリ本体
- `styles.css`: レスポンシブ対応の見た目
- `app.js`: ルーレット処理、Web Audio、250件のお題データ
- `.nojekyll`: GitHub Pages の素通し配信用

## Data

お題は合計250件です。

- 公式由来: 207件
- 追加作成: 43件

公式由来のお題は、以下の公開ページから抽出した「今週のお題」系の文言を利用しています。

- https://blog.hatenablog.com/archive/category/Odai
- https://staff.hatenablog.com/archive/category/%E4%BB%8A%E9%80%B1%E3%81%AE%E3%81%8A%E9%A1%8C
- https://blog.hatena.ne.jp/-/campaign/odai

GitHub Pages は静的ホスティングのため、公開後にブラウザから外部ページを直接スクレイピングせず、取得済みのお題リストを同梱しています。
